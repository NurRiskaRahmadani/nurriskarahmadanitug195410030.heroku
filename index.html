Belajar dasar-dasar web

Website menjadi salah satu pilihan para pebisnis online untuk menawarkan barang dagangan mereka. Banyak orang hanya mengetahui tentang halaman website, tapi ternyata ada beberapa hal tentang desain website yang tak banyak diketahui oleh kebanyakan orang. Website bertugas seperti pos, di mana orang dapat meninggalkan pesan dan saling bertukar komentar di halaman web tersebut. Sebelum masuk ke dalam sebuah web pun, pasti Anda menggunakan browser untuk mengakses halaman web

Desain-desain dasar dalam pembuatan website :

- HyperText Markup Language (HTML)

HyperText Markup Language/HTML adalah script dasar yang dipakai dalam pembuatan sebuah website yang bisa ditampilkan oleh browser. Untuk memberi tahu browser file teks berisi HTML, digunakan ekstensi file .html. Karena dokumen HTML tidak lain adalah file teks maka dapat menggunakan editor teks apa saja untuk membuatnya, misalnya Notepad, Microsoft Word.

-Cascading StyleSheet (CSS)

Cascading StyleSheet atau yang biasa disebut sebagai CSS. CSS menggunakan bahasa yang sangat kompleks. Kalau Anda menginginkan desain website yang indah, Anda harus mengenal HTML dan CSS dengan sangat baik yang bisa membantu anda dalam mendesain website anda nantinya. CSS itu adalah asesoris dari HTML, dia berfungsi agar desain web konsisten dan cantik di setiap halamannya. Misalkan kalau besaran judul nilainya 16 pixel, di setiap halaman web dapat dipastikan juga 16 pixel dengan dCSS. Tetapi kalau tiba-tiba ingin mengganti menjadi 18 pixel, anda tidak perlu merubah seluruh halaman web menjadi 18 pixel. CSS memungkinkan anda untuk mengganti satu halaman saja, maka ratusan halaman website akan berubah dengan sekejab. Itulah keuntungan dari CSS, mempermudah dan mempercepat kerja anda dalam mendesain web.

-Javascript

Pemrograman Javascript jauh lebih kompleks dibanding dengan CSS. Pembahasan mengenai Javascript bisa jadi menghabiskan satu buku sendiri saking kompleksnya. Javascript bertujuan untuk membuat desain website selain bisa berinteraksi, juga bisa menampilkan animasi-animasi menarik sehingga website anda bisa menjadi lebih hidup karena bergerak dan tentu mendapat perhatian pengunjung lebih banyak.

-Browser dan kode klien

Pengembangan desain website dilakukan dengan tiga bahasa pemrograman. HTML untuk penanda konten, CSS untuk desain tampilan, dan Javascript untuk interaktifitas. Ketika browser menerima ketiga perintah tersebut dari web server, maka browser akan menjalankan ketiganya. Beberapa browser utama yang kerap digunakan untuk mengakses beberapa website adalah Microsoft Internet Explorer, Mozilla Firefox, Google Chroome, dan Opera.
